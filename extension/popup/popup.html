Welcome to Syncr

Syncr needs a small local app to talk to Discord. This is a one-time setup — no admin rights needed.

  1. Download the host installer
  2. Run it when Windows asks
  3. Return here — we'll connect automatically

Install native host

  1. Click Download installer below
  2. In your Downloads folder, double-click Install-Syncr-Host.bat
  3. If Windows asks to run it → click Run
  4. If SmartScreen appears → More info → Run anyway
  5. Wait until the window says Done, press Enter, then click Connect below

Installs to your user folder only — no administrator password required.
